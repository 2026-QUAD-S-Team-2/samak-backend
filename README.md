### Key Features 
1. 통합 채용 공고 검증
- AI 기반 공고 신뢰도 분석 및 의심 패턴 탐지 및 분석 메시지 제공
- 유사 피해 사례 연계
- 해당 국가 평균 소득 관련 통계 제공
  
2. 사기 관련 정보 제공
- 사기 수법 및 대응 가이드 제공
- 최신 사기 뉴스 제공
- 사기 예방을 위한 데일리 퀴즈 제공

3. 피해 사례 통합 관리 시스템
- 직접 겪은 피해 사례 등록
- 국가, 회사명, 전화번호, 텔레그램 ID 등 식별자 기반 검색
